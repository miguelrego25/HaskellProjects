data Point a = Point { x :: a , y :: a , z :: a } deriving (Eq , Show )

:t uncurry (uncurry Point)
uncurry (uncurry Point) :: forall a. ((a, a), a) -> Point a

Resolução¶

Vamos começar por verificar o tipo de $in = \widehat{flip \ \widehat{Point}}$

$in = \widehat{flip \ \widehat{Point}}$

{ def. $flip$, $flip \ f = \overline{\widehat{f} . swap}$ }

$= \widehat{\overline{\widehat{\widehat{Point}} . swap}} $

{ uncurry . curry = id }

$= \widehat{\widehat{Point}} . swap $

Sabemos, pelo primeiro diagrama, que

$(A \times A) \times A \xrightarrow{\widehat{\widehat{Point}}} Point A$

Sabemos ainda, pelo segundo diagrama, que

$? \xrightarrow{\widehat{\widehat{Point}} . swap} Point A$

Ou seja, pela definição de $swap$ podemos então concluir que $? \ = \ A \times (A \times A)$.

E quanto a $out = \ ?$

Analisando os dois diagramas vemos que:

  • [Primeiro diagrama]: $Point A \xrightarrow{out = <<x,y>,z>} (A \times A) \times A$
  • [Segundo diagrama]: $Point A \xrightarrow{out=?} A \times (A \times A)$

Facilmente se conclui que no segundo diagrama $out = swap \ . <<x,y>,z> \ = \ <z, <x,y>>$, ou seja, temos finalmente:

  • [Primeiro diagrama]: $Point A \xrightarrow{out = <<x,y>,z>} (A \times A) \times A$
  • [Segundo diagrama]: $Point A \xrightarrow{out = <z, <x,y>>} A \times (A \times A)$
