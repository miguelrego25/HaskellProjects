$\def\cata#1{\mathopen{(\!|}#1\mathclose{|\!)}}\def\kons#1{\underline{#1}}$ $for \ b \ i = \cata{[\kons{i}, b]}$

{ propriedade universal, fazendo $k = for \ b \ i$ }

$for \ b \ i = \cata{[\kons{i}, b]} \equiv (for \ b \ i) \ . in = [\kons{i}, b] \ . (id + (for \ b \ i))$

{ def. in = [zero, succ] }

$\equiv (for \ b \ i) \ . [zero, succ] = [\kons{i}, b] \ . (id + (for \ b \ i))$

{ fusão-$+$, lei (20) ; absorção-$+$, lei (22) }

$\equiv [(for \ b \ i) \ . zero, (for \ b \ i) \ . succ] = [\kons{i} \ . id, b \ . (for \ b \ i)]$

{ natural-id, lei (1) ; eq-$+$, lei (27) }

$(for \ b \ i) \ . zero = \kons{i}$

$(for \ b \ i) \ . succ = b \ . (for \ b \ i)$

{ igualdade extensional, lei (71) }

$((for \ b \ i) \ . zero) \ n = \kons{i} \ n$

$((for \ b \ i) \ . succ) \ n = (b \ . (for \ b \ i)) \ n$

{ def-comp, lei (72) }

$for \ b \ i \ (zero \ n) = \kons{i} \ n$

$for \ b \ i \ (succ\ n) = b \ (for \ b \ i \ n)$

{ def. zero, def. succ, def-const, lei (74) }

$for \ b \ i \ 0 = i$

$for \ b \ i \ (n + 1) = b \ (for \ b \ i \ n)$
