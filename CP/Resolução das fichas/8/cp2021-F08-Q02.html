Resolução¶

$\def\cata#1{\mathopen{(\!|}#1\mathclose{|\!)}}\def\kons#1{\underline{#1}}$ Vamos então mostrar que $T \ f = \cata{in \ . B \ (f,id)}$ é a função map das sequências finitas.

Pela propriedade universal, fazendo $k = T \ f$, mostrar que $T \ f = \cata{in \ . B \ (f,id)}$ é o mesmo que mostrar que:

$T \ f . in = in \ . B \ (f,id) . F \ (T \ f)$

{ $T \ X = X^{\star}$ ; $T \ f = f^{\star}$ }

$f^{\star} . in = in \ . B \ (f,id) \ . F \ f^{\star}$

{ $B \ (f,id) = id + f \times id$ ; $F \ f^{\star} = B \ (id,f^{\star}) = id + id \times f^{\star}$ }

$f^{\star} . in = in \ . (id + f \times id) \ . (id + id \times f^{\star})$

{ $in = [\kons{[]},cons]$ }

$f^{\star} . [\kons{[]},cons] = [\kons{[]},cons] \ . (id + f \times id) . (id + id \times f^{\star})$

{ fusão-$+$; functor-$+$; functor-$\times$; $id^{\star} = id$ }

$[f^{\star} . \kons{[]},f^{\star} . cons] = [\kons{[]},cons] \ . ((id . id) + (f.id \times id . f^{\star})$

{ natural-id ; absorção-$+$ }

$[f^{\star} . \kons{[]},f^{\star} . cons] = [\kons{[]},cons \ . (f \times f^{\star})]$

{ eq-$+$ }

$f^{\star} . \kons{[]} = \kons{[]}$

$f^{\star} . cons = cons \ . (f \times f^{\star})$

{ def-const; def-comp ; pointwise ; $cons(h,t) = h:t$ }

$f^{\star} \ [] = []$

$f^{\star} (h:t) = f \ h \ : f^{\star} \ t$
