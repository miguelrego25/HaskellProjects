Resolução (i)¶

Mostrar que $B \ (X,Y) = X \times Y$ é bifunctor é o mesmo que mostrar que:

1. $B \ (id,id) = id$

{ $B (X,Y) = X \times Y$ }

$\equiv id \times id = id$

{ functor-id-$\times$ }

$\equiv id = id$

2. $B \ (f.g,h.k) = B \ (f,h) \ . B \ (g,k)$

{ $B (X,Y) = X \times Y$ }

$\equiv (f.g) \times (h.k) = (f \times h) \ . (g \times k)$

{ functor-$\times$ }

$\equiv (f \times h) \ . (g \times k) = (f \times h) \ . (g \times k)$

Resolução (ii)¶

Mostrar que $B \ (X,Y) = X + Y$ é bifunctor é o mesmo que mostrar que:

1. $B \ (id,id) = id$

{ $B (X,Y) = X + Y$ }

$\equiv id + id = id$

{ functor-id-$+$ }

$\equiv id = id$

2. $B \ (f.g,h.k) = B \ (f,h) \ . B \ (g,k)$

{ $B (X,Y) = X + Y$ }

$\equiv (f.g) + (h.k) = (f + h) \ . (g + k)$

{ functor-$+$ }

$\equiv (f + h) \ . (g + k) = (f + h) \ . (g + k)$

Resolução (iii)¶

Mostrar que $B \ (X,Y) = X + Y \times Y$ é bifunctor é o mesmo que mostrar que:

1. $B \ (id,id) = id$

{ $B (X,Y) = X + Y \times Y$ }

$\equiv id + id \times id = id$

{ functor-id-$\times$ }

$\equiv id + id = id$

{ functor-id-$+$ }

$\equiv id = id$

2. $B \ (f.g,h.k) = B \ (f,h) \ . B \ (g,k)$

{ $B (X,Y) = X + Y \times Y$ }

$\equiv (f.g) + ((h.k) \times (h.k)) = (f + (h \times h)) \ . (g + (k \times k))$

{ functor-$\times$ }

$\equiv (f.g) + ((h \times h) \ . (k \times k)) = (f + h \times h) \ . (g + k \times k)$

{ functor-$+$ }

$\equiv (f + (h \times h)) \ . (g . (k \times k)) = (f + (h \times h)) \ . (g + (k \times k))$
