$\def\cata#1{\mathopen{(\!|}#1\mathclose{|\!)}}\def\kons#1{\underline{#1}}\def\ana#1{\mathopen{[\!(}#1\mathclose{)\!]}}$ $T f = \cata{in . B \ (f,id)}$

{ universal-cata, lei (45) }

$T f \ . in = in \ . B \ (f,id) \ . F \ (T f)$

{ base-cata }

$T f \ . in = in \ . B \ (f,id) \ . B \ (id, T f)$

{ $B \ (f,id) \ . B \ (id, T f) = B \ (f \ . id, id \ . T f) = B \ (id \ . f, T f \ . id) = B \ (id,T f) \ . B \ (f,id)$ }

$T f \ . in = in \ . B \ (id, T f) \ . B \ (f,id)$

{ shunt-left + shunt-right }

$out \ . T f = B \ (id, T f) \ . B \ (f,id) \ . out$

{ base-ana }

$out \ . T f = F \ (T f) \ . B \ (f,id) \ . out$

{ universal-ana }

$T f = \ana{B \ (f,id) \ . out}$
