Resolução¶

$\def\cata#1{\mathopen{(\!|}#1\mathclose{|\!)}}\def\kons#1{\underline{#1}}\def\ana#1{\mathopen{[\!(}#1\mathclose{)\!]}}\def\repeat{\textit{repeat}}$ Parte I

$\repeat = \ana{<id,id>}$

{ universal-ana }

$out \ . \repeat = F \ \repeat \ . <id,id>$

{ $F f = id \times f$ }

$out \ . \repeat = (id \times \repeat) \ . <id,id>$

{ shunt-right ; absoro-$\times$ ; natural-id }

$\repeat = in \ . <id, \repeat>$

{ $in = cons$}

$\repeat = cons \ . <id, \repeat>$

{ pointwise }

$\repeat \ a = cons (a, \repeat \ a)$

{ $cons(h,t) = h:t$ }

$\repeat \ a = a : \repeat \ a$

Parte II

$map \ f \ . \repeat = \repeat \ . f$

{ $map \ f = T \ f$ ; $\repeat = \ana{<id,id>}$ }

$\equiv \ T \ f \ . \ana{<id,id>} = \ana{<id,id>} \ . f$

{ absoro-ana }

$\equiv \ \ana{B(f,id) \ . <id,id>} = \ana{<id,id>} \ . f$

{ $B(f,id) = f \times id$ }

$\equiv \ \ana{(f \times id) \ . <id,id>} = \ana{<id,id>} \ . f$

{ absoro-$\times$ }

$\equiv \ \ana{<f,id>} = \ana{<id,id>} \ . f$

{ fuso-ana }

$\equiv \ F \ f \ . <f,id> \ = \ <id,id> \ . f$

{ $F \ f = id \times f$ }

$\equiv \ (id \times f) \ . <f,id> \ = \ <id,id> \ . f$

{ absoro-$\times$ }

$\equiv \ <f,f> \ = \ <id,id> \ . f$

{ fuso-$\times$ }

$\equiv \ <f,f> \ = \ <f,f>$
