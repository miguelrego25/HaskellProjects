Resolução (a)¶

Queremos demonstrar que $[i_1, i_2] = id$

Partindo da propriedade universal e fazendo $k = id$ temos:

$id = [f,g]$

{ propriedade universal de $[f,g]$ }

$id \cdot i_1 = f \ ; \ id \cdot i_2 = g$

{ natural-id, lei (1) }

$i_1 = f \ ; \ i_2 = g$

{ propriedade universal de $[f,g]$ }

$id = [f,g] = [i_1,i_2]$

Resolução (b)¶

Queremos demonstrar que $< h, k > \cdot f \ = \ < h \cdot f, k \cdot g>$

Partindo da propriedade universal e fazendo $k \ = \ <h,k> \cdot f$ temos:

{ propriedade universal de $<f,g>$ }

$\pi_1 \cdot (<h,k> \cdot f) \ = \ h \cdot f$

$\ \pi_2 \cdot (<h,k> \cdot f) \ = \ k \cdot f$

{ assoc-comp, lei (2) }

$(\pi_1 \cdot <h,k>) \cdot f \ = \ h \cdot f$

$(\pi_2 \cdot <h,k>) \cdot f \ = \ k \cdot f$

{ cancelamento-$\times$, lei (7) }

$h \cdot f = h \cdot f$

$k \cdot f = k \cdot f$

{ prop. reflexiva da igualdade }
